專案名稱
多語言 Vue 3 前端展示範例

專案介紹
本專案為一個基於 Vue 3 建立的多語言前端應用示範，包含：

動態切換中文（繁體）與英文介面

帳號管理 (Account Settings)

推播裝置設定 (Push Notification Settings)

彈窗提示 (Modal) 與通知提示 (Toast) 元件

多語言翻譯檔案分離管理，方便新增多種語言

環境需求
Node.js (建議版本 >= 14)

npm (Node 套件管理工具)

支援現代瀏覽器的 Web 伺服器或 Vite 開發服務器

快速啟動
下載或克隆本專案

進入專案資料夾


cd D:\ibubble.ivlc.com\ibubble.ivlc.com\frontend(我用D槽,請自行更換)
安裝套件依賴


npm install
啟動本地開發伺服器


npm run dev
瀏覽器開啟 http://localhost:3000 或命令行展示的本地網址: http://localhost:5173/

功能展示
頁面頂部語言下拉可切換中英文

依語言即時改變介面文字

各類帳號管理操作示意，包括第三方帳號連結、本地帳號編輯、OTP 驗證等

推播設定語言、角色篩選選單

專案架構

frontend/
  src/
    components/           # Vue 組件 (AccountSettings、PushNotification、Modal、Toast...)
    i18n/                 # 多語言 JSON 檔 (zh-TW.json, en-US.json)
    App.vue               # 根組件，整合所有子組件並管理語言狀態
    main.js               # Vue 入口
  package.json            # 專案依賴管理
  vite.config.js          # Vite 配置檔（若有）
多語言管理
多語言文字由 src/i18n 下的 JSON 檔管理

使用 vue-i18n 實現國際化，方便後續擴充

文字全部通過翻譯函式 t('key') 取得，確保多語言正確切換

注意事項
本專案為靜態前端示範，所有帳號及 OTP 操作均為模擬，無真實後端連線

部分功能需未來整合 API 後方能完整實現動態資料
